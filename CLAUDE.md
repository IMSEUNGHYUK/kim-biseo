# 김비서 — 업무 대시보드

마케팅팀장을 위한 올인원 업무 관리 대시보드. 실시간 날씨, 시간 기반 할 일 관리, 매출 데이터 분석을 한 곳에서 처리합니다.

## 🎯 주요 기능

### 📋 오늘 할 일
- 시간(시·분) 순으로 자동 정렬
- 우선순위별 구분 (높음/보통/낮음)
- **인라인 편집** — 업무명과 시간 직접 수정 가능
- 완료 체크 및 삭제 기능
- localStorage에 자동 저장

### 🌅 아침 브리핑 (Morning Briefing)
- 매일 **첫 방문 시 자동 팝업**
- 실시간 서울 날씨 (Open-Meteo API)
- 우선순위 높은 할 일 5개
- 오늘 일정 (10:00, 14:00, 16:00 등)

### 📅 주간 일정 캘린더
- 월~일 7일간 일정 뷰
- 오늘(16일) 강조 표시
- 컬러 코딩된 이벤트

### 💰 매출 현황
- 총 매출, 최고 제품, 최고 지역 KPI
- 일별 매출 추이 라인 차트
- 제품별 매출 및 지역별 비중 시각화

### 📊 회의 액션 아이템
- 3월 16일 마케팅 회의 기반
- 담당자별 업무 및 마감 기한
- 팀 업무 현황 한눈에 파악

## 📁 파일 구조

| 파일명 | 역할 |
|--------|------|
| **업무대시보드.html** | 메인 대시보드 (할 일, 날씨, 일정, 매출, 차트) |
| 매출보고서.html | 매출 분석 리포트 (30건 거래 분석) |
| 매출차트.html | 매출 데이터 시각화 (6개 차트) |
| 마케팅업무흐름.html | 업무 프로세스 다이어그램 (기획→배포) |
| subtrac_분석보고서.html | SubTrackr 사이트 분석 |
| enoughbox_분석보고서.html | 이너프 이유식 사이트 분석 |
| 매출데이터.csv | 2026년 1-2월 매출 원본 데이터 |
| 회의록.txt | 3월 16일 마케팅팀 주간 회의 내용 |
| 행사포스터.jpg | AI 인프라 포럼 2026 행사 포스터 |

## 🛠 사용 기술

- **HTML5** — 시맨틱 마크업, 폼 요소 (시간 입력)
- **CSS3** — CSS 변수, Grid, Flexbox, 애니메이션
- **JavaScript (Vanilla)** — 
  - 시간순 정렬 알고리즘
  - localStorage 상태 관리
  - 인라인 편집 모드 토글
  - Open-Meteo API 연동 (날씨)
  - Chart.js 차트 렌더링
